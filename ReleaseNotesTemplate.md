Release Notes for Pre-Release of Version 1.0.0


===[ WARNING ]===

This software is being released in a "pre-alpha" state.  This means that 
the software may be unstable or unpredictable.  Please use this version
for testing and tinkering ONLY.  There is currently no way to "export"
data for backup and the software may render the your data unusable at any
time.

OK, reality check: it is unlikely that the software will lose your data and
crash your computer.  But it is VERY likely that the software may not work
properly or have a complete set of usable features.  This is being released
because we feel that all of the programs major features are in place but 
that they need testing in order to be fixed and polished.

Please remember that this software is currently being released FOR TESTING ONLY.


===[ Installation ]===

Please follow installation instructions at:

http://code.google.com/p/cropplanning/wiki/SetupAndInstallation

Briefly: 1) Make sure Java 1.5 or greater is installed, 2) Open the archive (which you had to do to read this), 3) Copy the enclosed files to your hard drive, 4) double click on the  program file (On Mac: CropPlanning, on Windows: CropPlanning.exe)

===[ Using the Program ]===

There is a brief introduction to the program available at

http://code.google.com/p/cropplanning/wiki/ProgramUsageIntroduction
